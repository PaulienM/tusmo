# TUSMO

## Les mots

Les mots ont été récupérés sur le site https://www.freelang.com/dictionnaire/dic-francais.php.
Pour filtrer les mots (suppression des noms propres et mots composés)

```bash
bin/filters.sh public/words.txt > public/filtered-word.txt
bin/filters.sh public/words.txt 6 > public/filtered-word-6.txt
bin/filters.sh public/words.txt 7 > public/filtered-word-7.txt
bin/filters.sh public/words.txt 8 > public/filtered-word-8.txt
bin/filters.sh public/words.txt 9 > public/filtered-word-9.txt
bin/filters.sh public/words.txt 10 > public/filtered-word-10.txt
bin/filters.sh public/words.txt 11 > public/filtered-word-11.txt
bin/filters.sh public/words.txt 12 > public/filtered-word-12.txt
bin/filters.sh public/words.txt 13 > public/filtered-word-13.txt
bin/filters.sh public/words.txt 14 > public/filtered-word-14.txt
```
